# Portfolio Project

A modern portfolio website built with [Next.js](https://nextjs.org/), [React](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Radix UI](https://www.radix-ui.com/).

## Features

- **Next.js 16**: Utilizing the latest features from Next.js for server-side rendering, routing, and performance optimization.
- **Tailwind CSS 4**: Modern utility-first CSS framework for rapid and customizable styling.
- **Radix UI Primitives**: Accessible and unstyled components for building a flexible UI system.
- **TypeScript**: Written in TypeScript for enhanced type safety and developer experience.

## Getting Started

First, make sure you have [Node.js](https://nodejs.org/) installed, then install the project dependencies:

```bash
# Install dependencies using npm
npm install

# Or using pnpm (since a pnpm-lock.yaml is present)
pnpm install
```

### Running the Development Server

Run the development server to view the application locally:

```bash
# Using npm
npm run dev

# Using pnpm
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

In the project directory, you can run:

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm start`: Runs the built app in production mode.
- `npm run lint`: Runs ESLint to check for code quality issues.
